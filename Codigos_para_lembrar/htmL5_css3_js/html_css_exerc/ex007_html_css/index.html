Principais Formatações

Negrito

A tag < b > não é mais usada para negrito

Usamos as tag semânticas como < strong > , que significa força. Possui significado e não só forma.

Para itálico use <em>para enfase ou itálico semantico.

Texto Marcado

Criação de marcação com a tag <mark>. A cor da marcação deve ser feeita com paramatro style dentro da tag <mark style="background-color: green;" >

Textos grandes e pequenos

Texto deletado e texto inserido

Para Hospedar os arquivos não podem ser muito grandes para nao comprometer os gastos com banda e tráfego(demora no carregamento).

ex: arquivo de video de 11,5MB, visitado por 1000 pessoas: 11,5 x 1000 = 11500 MB ou 11500/1024 =11,2GB de consumo de banda - carrerrimo.

Baixar HandBrake para tratar os vídeos.
Não convert para ogv, buscar outro conversor, 
para dacr opções para os browser´s.

Nunca hospedar videos com grande qualidade 1080p ou 4k

Uma possibilidade é mandar para youtube ou vimeo e incorporar com <iframe> </iframe>

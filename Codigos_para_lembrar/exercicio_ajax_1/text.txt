Trata-se de requisição XMLHttpRequest(), metodologia AJAX.

Para rodar deve ter um servidor apache instalado e rodando.

A página principal (index.html), é a base. Nela você vai fazer as chamadas para as páginas requisitadas.

A página conteúdo.html é não formatada e fornece o texto requisitado.

A Requisiçãi é feita atravéz de Java Script ajax.js

O CSS é interno.

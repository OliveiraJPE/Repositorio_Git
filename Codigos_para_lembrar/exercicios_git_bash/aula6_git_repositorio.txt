CRIAR UM REPOSITÓRIO NOVO
//snapshot = instataneo

peoj2@LAPTOP-DRDIEMQ2 MINGW64 ~ (master)
$ pwd                     //verificar onde estamos
/c/Users/peoj2


peoj2@LAPTOP-DRDIEMQ2 MINGW64 ~ (master)
$ mkdir Repositorio_Git_Boson

peoj2@LAPTOP-DRDIEMQ2 MINGW64 ~/Repositorio_Git_Boson (master)
$ pwd
/c/Users/peoj2/Repositorio_Git_Boson

peoj2@LAPTOP-DRDIEMQ2 MINGW64 ~/Repositorio_Git_Boson (master)
$ git init                      //iniciar o repositório
Initialized empty Git repository in C:/Users/peoj2/Repositorio_Git_Boson/.git/

peoj2@LAPTOP-DRDIEMQ2 MINGW64 ~/Repositorio_Git_Boson (master)
$ ls -a                             //mostra o conteúdo inclusive arquivos ocultos como o .git
./  ../  .git/

peoj2@LAPTOP-DRDIEMQ2 MINGW64 ~/Repositorio_Git_Boson (master)
$ start notepad++ projeto.txt                           // criar um projeto direto com notepad ++.
                                                       // poderia usar $touch projeto.txt dentro do diretório tambem

peoj2@LAPTOP-DRDIEMQ2 MINGW64 ~/Repositorio_Git_Boson (master)
$ git add projeto.txt                       //adicionar a staging area área de preparação



peoj2@LAPTOP-DRDIEMQ2 MINGW64 ~/Repositorio_Git_Boson (master)
$ git commit -m "Projeto iniciado"
[master (root-commit) a37babf] Projeto iniciado
 1 file changed, 1 insertion(+)
 create mode 100644 projeto.txt

peoj2@LAPTOP-DRDIEMQ2 MINGW64 ~/Repositorio_Git_Boson (master)
$ git status
On branch master
nothing to commit, working tree clean


CRIAR MAIS DOIS ARQUIVOS

peoj2@LAPTOP-DRDIEMQ2 MINGW64 ~/Repositorio_Git_Boson (master)
$ touch tarefa.txt

peoj2@LAPTOP-DRDIEMQ2 MINGW64 ~/Repositorio_Git_Boson (master)
$ touch pendencias.txt


ADICIONAR TODOS OS NOVOS ARQUIVOS A STAGING AREA

peoj2@LAPTOP-DRDIEMQ2 MINGW64 ~/Repositorio_Git_Boson (master)
$ git add *

peoj2@LAPTOP-DRDIEMQ2 MINGW64 ~/Repositorio_Git_Boson (master)
$ git commit -m "Novos arquivos adicionados"
[master ccf51a2] Novos arquivos adicionados
 2 files changed, 0 insertions(+), 0 deletions(-)

 create mode 100644 tarefa.txt

peoj2@LAPTOP-DRDIEMQ2 MINGW64 ~/Repositorio_Git_Boson (master)
$ git status
On branch master
nothing to commit, working tree clean


RENOMEAR UM DOS ARQUIVOS

peoj2@LAPTOP-DRDIEMQ2 MINGW64 ~/Repositorio_Git_Boson (master)
$ mv pendencias.txt restante.txt

peoj2@LAPTOP-DRDIEMQ2 MINGW64 ~/Repositorio_Git_Boson (master)

Dimensionamento das Instalações elétricas residênciais


Entenda melhor como dimensionar condutores e disjuntores e depois divirta-se com o aplicativo.



Encontre o Disjuntor e o Condutor para seu equipamento.

OBS.: Condutor máximo 25mm². Neste caso não há a relatório final. Você deve anotar o resultado e repetir a operação todas as vezes que necessitar.

Nome do Circuito.


Coloque "1" para circuito de iluminação, e "2" para circuito de força.


Potência em VA


Tensão em Volts: 1F + N = 127V, 3F+N = 220V, 2F + N = 127V, 2F = 220V, 3F = 220V.


Insira "1" para Monofásico, "2" para Bifásico, "3" para bifásico + neutro, "4" para trifásico ou "5" para trifásico + Neutro.


Distancia entre carga e quadro de luz e/ou força em metros.


Fator de correção K1 - Fator de correção de temperatura - NBR 5410. Ex.:: Aplicar 1,12 para 20ºC, 1,06 para 25ºC, 0,94 para 35ºC ou 0,87 para 40ºC.


Fator de correção K2 - Fator de correção de agrupamento - NBR 5410. Ex.:: Aplicar 0,8 para 02 circuitos agrupados, 0,7 para 03 circuitos agrupados, 0,65 para 04 circuitos agrupados...


Fator de correção K3 - Fator de correção de resistividade do solo - NBR 5410. Não aplicável a residências - Manter 1.
